qué es mozilla? ..

Qué es mozilla? Mozilla es quién está detrás del desarrollo de Firefox y Thunderbird, pero eso es lo único que hace? qué es lo que lo motiva a hacerlo?

Promote openness, innovation and opportunity on the Internet

Mozilla es una organización sin fines de lucro, que busca promover la apertura, la innovación y la oportunidad en Internet.

comunidad global

Mozilla es una comunidad global, de millones de personas alrededor del mundo quienes creen en el proyecto y colaboran activamente en él.

Si bien mayormente cuando se habla acerca de Mozilla se les viene a la cabeza Firefox. Mozilla es mucho más que Firefox.

webmakers

https://webmaker.org

Mozilla Webmaker es un projecto que busca transformar a los usuarios comunes (que son simples consumidores) en creadores. Se enseña a conceptos básicos de como editar una web, como publicar y combinar un video con contenido en linea y algunas maneras de como mantenerse seguro y pasar un buen rato en la web

aprender haciendo

https://thimble.webmaker.org/en-US/projects/map/edit

Mozilla Thimble es una manera simple e interactiva de aprender HTML. En la parte izquierda tienes un editor y en la parte derecha puedes ver inmediatamente el resultado del código que vas colocando. Puedes escoger entre los proyectos existentes y escribir sobre ellos. El editor te muestra cuando has cometido algún error y te muestra el link a la comentación en MDN.

Lights, Camera, popcorn!

http://mozillapopcorn.org

Mozilla Popcorn está pensado para crear experiencias web interactivas mezclando videos y audio. El proyecto cuenta además con un editor que te permite superponer contenido web sobre los videos e e interactuar con partes de la página.

creado para desarrolladores ..

MDN es el portal de documentación técnica de mozilla, donde se crean recursos y demos para la web

... we need you

Photo by Paul Downey
Cualquiera puede contribuir, no existe algún prerrequisito previo. Ya sea traduciendo, agregando mejor contenido, etc. Juntos podemos seguir impulsando la innovación en la Web para servir a un bien mayor. Se inicia aquí, contigo.

un nuevo desafío

Photo by Jason Weaver
One very obvious change in the last years was that the web became much more mobile. Instead of desktop machines we use laptops, smart phones and tablets and all of these devices get more powerful by the month. They all come with different resolutions and connectivity issues, though and Mozilla is happy to take on the challenge of moving the open web from immobile machines to on-the-go computing.

html5 ftw!??

estado actual

Photo by Michael Gil
However, when native apps took off a lot of companies realised that there is money to be made by not keeping things open. Apps can do more than HTML5 can because the OS and hardware providers grant them more access and control the experience. If you control the software and the hardware, you can build great, but very bespoke products. If you allow for open technologies to be used, you lose control and you make less money as a hardware provider. As a developer, you shouldn't have to go that route.

Hardware lockout

Photo by Doug Geisler
The most frustrating part about this is that phone and operating system providers lock out browsers and JavaScript. The fun parts to use in a smartphone are the camera, the sensors and to use it to send messages to your friends and call people. To do this, however you'll need to write native code. Browsers, in general are locked out. This is not fair when you claim at the same time that HTML5 is the platform to bring a great future.

esto no es muy abierto ..

Mozilla está trabajando en definir e implementar APIs js (open APIs) que darán a los desarrolladores web acceso al hardware de los dispositivos. Son estándares, de manera que pueden ser implementadas en otros navegadores.

Web API work

http://arewemobileyet.com

At arewemobileyet.com we have a live view of how far down the line we are in defining and implementing open APIs that will give web developers access to this hardware. These are standards that can be used by other browser vendors and are implemented in Firefox. As you can see we are very far down the line, the only thing missing still is USB and NFC connectivity, other than that all that is in your phone can be accessed by JavaScript.

Open Web Device

http://openwebdevice.com

Es una iniciativa de Telefonica por llevar la web a las personas que no pueden permitirse tener un smartphone de gama alta, pero que deberían tener la oportunidad de tener una buena experiencia con sus móviles.

Powered by Firefox OS

Firefox OS, el nevo sistema para móviles desarrollado por Mozilla.
Photo by Alesa Dam
Firefox OS se basa en tres componentes: Gonk es la capa más baja de b2g, es un kernel linux que se relaciona con el hardware de nuestro dispositivo celular Gecko es el motor de renderizado (el mismo que usa Firefox) Gaia es la interfaz - todo lo que se ve - escrito en HTML5

Interface flexibility

Toda la interfaz de usuario de Firefox OS está escrita con tecnologías web. Esto significa que la customización de será mucho más sencilla, todo es HTML, CSS y JS.

Como probar Firefox OS…

  • GAIA en Firefox Nightly (simplest way)
  • desktop simulator (OSX / Linux)
  • desktop builds (OSX / Linux / Windows)
  • in your own device (more hard, more funny :D)

https://wiki.mozilla.org/Gaia/Hacking

You can run Firefox OS on your own computer inside Firefox Nightly or with Desktop simulators. The easiest is to run it in the browser and build your apps for it with the environment you are familiar with. You can put it on your own phones but the process is not easy (yet) The idea of Firefox OS is to bring new, affordable web-enabled phones to people and not to replace Android/iOS.

Existen varias maneras de mantenerte al día con Firefox OS/Boot2Gecko. Están son las principales.

Persona - sé dueño de tu identidad

http://www.mozilla.org/en-US/persona/

Mozilla Persona es una manera de logearse en sitios y aplicaciones web. En vez de tener que crear un usuario y password para cada producto te puedes logear con Persona, esto te permitirá identificarte con alguno de las cuentas emails que tengas verificadas. De esta manera no tienes miles de cuentas y además solo le das a los sitios la información que deseas, en vez de todos tus datos.

no seria divertido sin APPS

https://marketplace.mozilla.org/

Un smartphone sin aplicaciones no sería muy interesante, por ello Mozilla está implementando una tienda de aplicaciones (Marketplace) que contendrá apps que puedes usar en celular con Firefox pero también funcionarán en el navegador.

Qué puedo hacer?

  • Code - Crear apps, contribuye a Gaia, contribute a B2G(Firefox OS)
  • Use - Usa el Marketplace, Usa las Web APIs, Use B2G(Firefox OS)
  • Feedback - Da tu opinión sobre las APIs, ayuda a buscar nuevas características, testea y registra bugs
Firefox OS necesitá más desarrolladores que ayuden a mejorar el proyecto. Puedes colaborar de diferentes maneras.
